新的工作室网站！

碳酸天剑
2022-11-28 (Updated: 2023-01-02)

答舟工作室 的原网站（WordPress 搭建的）已经废弃多时，需要一个新的网站。

动态站还是静态站呢？
我觉得，搭建一个静态站用 GitHub Page 挂在 GitHub 上比较符合答舟工作室现在的状态：一个松散的挂名组织，所有活动仅限线上。

本站使用我熟悉的博客框架 Hexo 搭建，主题选用了 hexo-theme-cactus。
